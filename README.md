# LJS 419 — *Erbario* (Illustrated Herbal of the Veneto, 15th c.)

A page-by-page reading **edition** of manuscript **Penn LJS 419**: full diplomatic
transcription of every legible name and note, an English translation of the notes,
and a modern botanical identification for each plant, with per-entry confidence grades.


## Source manuscript
University of Pennsylvania, Kislak Center, Lawrence J. Schoenberg Collection,
**Oversize LJS 419**. Facsimile on OPenn / BiblioPhilly.
